# Getting started

## Installation

### With Cocoapods

1. Add the following entry to your Podfile:
```swift
pod ‘VideoUploaderIos’
```
3. Then run `pod install`
4. Don’t forget to import ‘VideoUploaderIos’ in every file you’d like to use api.video video uploader library
